O que é JavaScript?

O JavaScript é uma linguagem de programação amplamente utilizada para desenvolver aplicações web interativas. Ela é executada no navegador do usuário, permitindo criar desde efeitos simples até aplicações web complexas.

Originalmente criado para adicionar interatividade às páginas HTML estáticas, o JavaScript evoluiu significativamente e agora é usado tanto no front-end quanto no back-end (com o auxílio do Node.js).

Como o JavaScript Roda no Navegador?

Todos os navegadores modernos possuem um motor JavaScript embutido que interpreta e executa o código JavaScript presente nas páginas web. Por exemplo, o Google Chrome utiliza o motor V8, enquanto o Firefox utiliza o SpiderMonkey.

Quando você visita uma página web que contém JavaScript, o navegador baixa o código e o executa localmente, permitindo interações dinâmicas sem a necessidade de recarregar a página.

Programando JavaScript Diretamente no Navegador

Você pode experimentar e executar código JavaScript diretamente no seu navegador usando as ferramentas de desenvolvedor. Isso é útil para testes rápidos ou para aprender os fundamentos da linguagem.

Para acessar o console JavaScript do navegador:

  1. Abra o navegador (Chrome, Firefox, Edge, etc.).
  2. Pressione F12 ou Ctrl + Shift + I (Cmd + Option + I no Mac) para abrir as ferramentas de desenvolvedor.
  3. Clique na aba Console.
  4. Você pode digitar código JavaScript diretamente no console e pressionar Enter para executá-lo.

Exemplo:


        console.log('Olá, mundo!');

Ao executar o código acima no console, você verá a mensagem Olá, mundo! exibida no console.

Ambientes Online para Programar em JavaScript

Além das ferramentas locais, existem ambientes online que permitem escrever e executar código JavaScript diretamente no navegador, sem a necessidade de instalar nada. Alguns dos mais populares são:

  • CodePen: Uma plataforma online para escrever e compartilhar código HTML, CSS e JavaScript.
  • JSFiddle: Um editor online para testar e compartilhar snippets de código JavaScript.
  • Repl.it: Um ambiente de desenvolvimento online que suporta várias linguagens, incluindo JavaScript.

Esses ambientes são excelentes para praticar, testar ideias rapidamente e compartilhar código com outras pessoas.

Ferramentas Necessárias

Embora seja possível programar diretamente no navegador ou em ambientes online, é recomendável configurar um ambiente local para projetos mais complexos. As principais ferramentas que você precisará instalar são:

  • Visual Studio Code
  • Node.js
  • Git

Instalando o Visual Studio Code

O Visual Studio Code, também conhecido como VSCode, é um editor de código-fonte poderoso e extensível. Vamos seguir os passos para instalá-lo:

  1. Acesse o site oficial do VSCode através do link acima e baixe a versão adequada para o seu sistema operacional.
  2. Siga as instruções de instalação padrão do seu sistema operacional.
  3. Após a instalação, abra o VSCode para verificar se tudo está funcionando corretamente.

Instalando Node.js

O Node.js permite executar JavaScript fora do navegador, o que é fundamental para o desenvolvimento de aplicações modernas.

  1. Acesse o site oficial do Node.js através do link acima e baixe a versão LTS (Long Term Support) recomendada.
  2. Siga as instruções de instalação padrão do seu sistema operacional.
  3. Após a instalação, abra o terminal ou prompt de comando e verifique a instalação executando: 
    node -v
    e 
    npm -v
    Esses comandos devem retornar as versões instaladas do Node.js e do npm.

Instalando Git

O Git é essencial para o controle de versões e colaboração em projetos de desenvolvimento.

  1. Acesse o site oficial do Git através do link acima e baixe a versão adequada para o seu sistema operacional.
  2. Siga as instruções de instalação padrão do seu sistema operacional.
  3. Após a instalação, abra o terminal ou prompt de comando e verifique a instalação executando: 
    git --version
    Este comando deve retornar a versão instalada do Git.

Configurando Extensões no VSCode

Para otimizar seu ambiente de desenvolvimento no VSCode, é recomendável instalar algumas extensões que facilitarão o trabalho com JavaScript:

  • JavaScript (ES6) code snippets
  • Live Server
  • Prettier - Code formatter

Para instalar essas extensões:

  1. Abra o VSCode.
  2. Vá até a aba de Extensões clicando no ícone de quadrados no menu lateral ou pressionando Ctrl + Shift + X.
  3. Pesquise pelo nome da extensão desejada e clique em Instalar.

Configurando o VSCode

Após instalar as ferramentas e extensões, vamos configurar o VSCode para otimizar seu ambiente de desenvolvimento. Para isso, iremos ajustar o arquivo de configurações do VSCode:

Abra o arquivo de configurações do VSCode pressionando Ctrl + , ou navegando até File > Preferences > Settings. Em seguida, adicione ou atualize as seguintes configurações no seu arquivo settings.json: 


            {
              "editor.tabSize": 2,
              "editor.fontFamily": "'Fira Code', Consolas, 'Courier New', monospace",
              "editor.fontLigatures": true,
              "editor.fontSize": 18,
              "editor.lineHeight": 27,
              "editor.letterSpacing": 0.5,
              "editor.defaultFormatter": "esbenp.prettier-vscode",
              "editor.formatOnSave": true,
              "prettier.singleQuote": true,
              "[html]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode"
              },
              "[css]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode"
              },
              "[json]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode"
              },
              "[javascript]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode"
              },
              "workbench.activityBar.location": "hidden",
              "liveServer.settings.donotShowInfoMsg": true
            }

Essas configurações fazem o seguinte:

  • Tab Size: Define o tamanho da tabulação para 2 espaços.
  • Font Family: Define a fonte usada no editor, incluindo ligaduras para melhor visualização do código (tenha sua fonte baixada).
  • Font Ligatures: Ativa ligaduras de fonte, que melhoram a legibilidade do código.
  • Font Size: Define o tamanho da fonte para 18px.
  • Line Height: Define a altura da linha para 27px.
  • Letter Spacing: Define o espaçamento entre letras para 0.5px.
  • Default Formatter: Define o Prettier como formatador padrão para o código.
  • Format On Save: Formata automaticamente o código ao salvar.
  • Single Quote: Configura o Prettier para usar aspas simples.
  • Activity Bar Location: Oculta a barra de atividades lateral.
  • Live Server Settings: Oculta mensagens informativas do Live Server.

Verificando a Configuração

Para garantir que todas as ferramentas estejam corretamente instaladas e configuradas, siga os passos abaixo:

  1. Abra o VSCode e crie um novo arquivo HTML (.html).
  2. Ao pressionar "!" o VSCode irá lhe auxiliar e criar a sintaxe inicial do HTML (falaremos disso em outras aulas).
  3. Dentro da tag <body>, crie uma tag <script>, onde você pode digitar um código como: 
    
                function saudacao(nome) {
                  return `Olá, ${nome}!`;
                }
                
                console.log(saudacao('Matheus'));
  4. Isso deixará o seu código assim: 
    
                <!DOCTYPE html>
                <html lang="en">
                  <head>
                    <meta charset="UTF-8" />
                    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                    <title>Document</title>
                  </head>
                  <body>
                    <script>
                      function saudacao(nome) {
                        return `Olá, ${nome}!`;
                      }
                
                      console.log(saudacao('Matheus'));
                    </script>
                  </body>
                </html>
  5. Salve o arquivo pressionando Ctrl + S. O Prettier deve formatar automaticamente o código conforme as configurações definidas.
  6. Inicie o Live Server clicando com o botão direito no arquivo HTML correspondente e selecionando Open with Live Server. Uma nova aba do navegador será aberta, exibindo sua página.
  7. E verificando o console do navegador você verá a mensagem que escreveu.

Se todos os passos acima forem concluídos sem erros, seu ambiente está pronto para iniciar o desenvolvimento de projetos em JavaScript!
