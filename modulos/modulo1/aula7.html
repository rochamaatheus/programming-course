Introdução aos Objetos

Em JavaScript, um objeto é uma coleção de dados e funcionalidades relacionadas, representadas por propriedades e métodos. Objetos permitem organizar e estruturar o código de forma eficiente.

Criando e Acessando Objetos

Podemos criar um objeto utilizando chaves {} e definir pares de chave e valor para representar suas propriedades.

Exemplo de Objeto Simples


var pessoa = {
  nome: 'Matheus',
  idade: 18,
  profissao: 'programador',
  possuiFaculdade: false,
};

console.log(pessoa.nome); // 'Matheus'
console.log(pessoa.possuiFaculdade); // false

As propriedades e métodos de um objeto são acessados utilizando a notação de ponto ..

Métodos em Objetos

Um método é uma função associada a uma propriedade de um objeto. Isso permite que o objeto execute ações com seus próprios dados.

Exemplo com Métodos


var quadrado = {
  lados: 4,
  area: function(lado) {
    return lado * lado;
  },
  perimetro: function(lado) {
    return this.lados * lado;
  },
};

console.log(quadrado.lados); // 4
console.log(quadrado.area(5)); // 25
console.log(quadrado.perimetro(5)); // 20

Note o uso da palavra-chave this para referenciar propriedades do próprio objeto dentro dos métodos.

Sintaxe Abreviada de Métodos

A partir do ES6, podemos definir métodos de forma abreviada, omitindo a palavra-chave function.

Exemplo com Sintaxe Abreviada


var quadrado = {
  lados: 4,
  area(lado) {
    return lado * lado;
  },
  perimetro(lado) {
    return this.lados * lado;
  },
};

Organizando o Código com Objetos

Objetos são excelentes para organizar o código em partes reutilizáveis e lógicas. Muitos objetos nativos do JavaScript, como Math e console, seguem esse padrão.

Exemplo com Objeto Math


console.log(Math.PI); // 3.141592653589793
console.log(Math.random()); // Número aleatório entre 0 e 1

var pi = Math.PI;
console.log(pi); // 3.141592653589793

Criando um Objeto

Um objeto é criado utilizando as chaves {}. Podemos atribuir propriedades e métodos conforme necessário.

Exemplo de Criação de Objeto


var carro = {};
var pessoa = {};

console.log(typeof carro); // 'object'

Acessando e Modificando Propriedades

Utilizamos a notação de ponto para acessar e modificar as propriedades de um objeto.

Acessando Propriedades


var menu = {
  width: 800,
  height: 50,
  backgroundColor: '#84E',
};

var bg = menu.backgroundColor;
console.log(bg); // '#84E'

Modificando Propriedades


menu.backgroundColor = '#000';
console.log(menu.backgroundColor); // '#000'

Adicionando Novas Propriedades e Métodos

Podemos adicionar novas propriedades e métodos a um objeto a qualquer momento. 


menu.height = 50;
menu.position = 'fixed';

console.log(menu);

A Palavra-chave this

A palavra-chave this faz referência ao objeto atual, permitindo acessar suas propriedades e métodos internamente.

Exemplo com this


var height = 120;
var menu = {
  width: 800,
  height: 50,
  metadeHeight() {
    return this.height / 2;
  },
};

console.log(menu.metadeHeight()); // 25

Sem o uso de this, o JavaScript buscaria a variável height no escopo global, o que poderia causar conflitos ou resultados inesperados.

Protótipo e Herança

Em JavaScript, objetos herdam propriedades e métodos de um protótipo, que é o objeto usado como base para sua criação. Isso permite reutilizar código e adicionar funcionalidades adicionais.

Exemplo de Herança


var menu = {
  width: 800,
};

console.log(menu.hasOwnProperty('width')); // true
console.log(menu.hasOwnProperty('height')); // false

O método hasOwnProperty é herdado do protótipo Object. Isso significa que o objeto menu tem acesso a métodos que não foram definidos diretamente nele, mas que estão disponíveis no seu protótipo.

Tudo é Objeto

Em JavaScript, quase tudo é um objeto. Isso inclui strings, números, booleanos, arrays, funções e até mesmo outros objetos. Isso ocorre porque esses tipos de dados herdam propriedades e métodos de seus protótipos.

Strings

Strings possuem métodos e propriedades que permitem manipulações avançadas. 


var nome = 'André';

console.log(nome.length); // 5
console.log(nome.charAt(1)); // 'n'
console.log(nome.replace('ré', 'rei')); // 'Andrei'
console.log(nome); // 'André'

Uma string herda propriedades e métodos do construtor String().

Números

Números também possuem métodos e propriedades, embora sejam tipos primitivos. Isso acontece devido à coerção, onde o JavaScript envolve o número em um objeto temporariamente. 


var altura = 1.8;

console.log(altura.toString()); // '1.8'
console.log(altura.toFixed()); // '2'

Funções

Funções são objetos de primeira classe em JavaScript, o que significa que podem ter propriedades e métodos. 


function areaQuadrado(lado) {
  return lado * lado;
}

console.log(areaQuadrado.toString());
// "function areaQuadrado(lado) {
//   return lado * lado;
// }"

console.log(areaQuadrado.length); // 1

Elementos do DOM

Elementos HTML selecionados no DOM também são objetos e possuem propriedades e métodos que permitem interagir com a página. 



≤a class="btn">Clique≤/a>
          

var btn = document.querySelector('.btn');

btn.classList.add('azul'); // Adiciona a classe 'azul'
console.log(btn.innerText); // 'Clique'
btn.addEventListener('click', function() {
  console.log('Clicou');
});

A maioria das interações em uma página web é realizada manipulando propriedades e métodos desses objetos do DOM.

Objetos Revolucionaram a Programação

A programação orientada a objetos (POO) permite que desenvolvedores criem aplicações mais organizadas, reutilizáveis e fáceis de manter. Em JavaScript, essa abordagem é amplamente utilizada, especialmente com o uso de Web APIs que permitem a interação entre JavaScript e o navegador.
