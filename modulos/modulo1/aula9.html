Introdução

Nesta aula, abordaremos Comentários, Operadores de Atribuição, o Operador Ternário e o If Abreviado em JavaScript. Vamos explorar cada um desses tópicos com exemplos práticos.

Comentários em JavaScript

Comentários são essenciais para documentar o código, tornando-o mais compreensível para outros desenvolvedores e para você mesmo no futuro. Existem dois tipos de comentários em JavaScript:

Comentário de Uma Linha

// Este é um comentário de uma linha

Comentário de Múltiplas Linhas


/*
  Este é um comentário
  de múltiplas linhas
  em JavaScript
*/

Nota: Comentar linhas de código pode ser útil para testes, mas evite deixar código comentado no arquivo final para manter a limpeza e eficiência do código.

Operadores de Atribuição

Operadores de atribuição permitem realizar operações matemáticas de forma mais concisa. Eles combinam uma operação com a atribuição do resultado à variável. Vamos ver alguns exemplos:

Exemplos de Operadores de Atribuição


var saldo = 200;
var deposito = 50;

// Adiciona deposito ao saldo
saldo += deposito; // saldo = saldo + deposito (250)

// Subtrai depósito do saldo
saldo -= deposito; // saldo = saldo - deposito (200)

// Multiplica saldo por um fator
saldo *= 2; // saldo = saldo * 2 (400)

// Divide saldo por um divisor
saldo /= 4; // saldo = saldo / 4 (100)

// Obtém o resto da divisão do saldo
saldo %= 30; // saldo = saldo % 30 (10)

// Eleva saldo a uma potência
saldo **= 3; // saldo = saldo ** 3 (1000)

Operador Ternário

O operador ternário é uma forma abreviada de escrever condicionais simples. Ele segue a sintaxe: 

condição ? expressão1 : expressão2;

Vamos ver um exemplo prático:

Exemplo de Operador Ternário


var nota = 85;
var resultado = (nota >= 70) ? 'Aprovado' : 'Reprovado';
console.log(resultado); // Aprovado

Este operador é especialmente útil quando precisamos atribuir valores a variáveis com base em uma condição.

If Abreviado

Quando a estrutura condicional possui apenas uma linha de código a ser executada, podemos omitir as chaves `{}` para tornar o código mais conciso. Veja o exemplo abaixo:

Exemplo de If Abreviado


var possuiVaga = false;

if(possuiVaga) console.log('Vaga disponível');
else console.log('Vaga indisponível');

// Ou, em outra linha
if(possuiVaga)
  console.log('Vaga disponível');
else
  console.log('Vaga indisponível');

Dica: A segunda opção, onde cada parte do if e else está em uma linha separada, melhora a legibilidade do código.
