Introdução

Na aula anterior, falamos sobre como em JavaScript "Tudo é Objeto", e mencionamos brevemente o DOM. Alguns alunos expressaram dificuldades em compreender esse conceito e como ele se relaciona com os objetos em JavaScript. Nesta aula de apoio, vamos explorar o DOM de forma mais detalhada, entendendo seu papel fundamental na interação entre JavaScript e as páginas web.

O que é o DOM?

O Document Object Model (DOM) é uma interface de programação que representa documentos HTML e XML como uma árvore de objetos. Isso permite que linguagens de programação, como JavaScript, acessem e modifiquem o conteúdo, estrutura e estilo de uma página web.

A Estrutura do DOM

Imagine o DOM como uma árvore genealógica, onde cada elemento HTML é um nó nessa árvore. Essa estrutura hierárquica permite navegar pelos elementos e acessar suas propriedades e métodos.

Relacionamento entre o DOM e Objetos em JavaScript

Em JavaScript, quase tudo é um objeto, incluindo elementos do DOM. Cada elemento HTML no DOM é representado como um objeto que possui propriedades e métodos que podemos acessar e manipular. Isso nos permite, por exemplo, alterar o texto de um parágrafo, modificar atributos e muito mais.

Acessando Elementos do DOM com JavaScript

Para manipular o DOM, primeiro precisamos acessar os elementos desejados. JavaScript fornece diversos métodos para selecionar elementos:

Usando getElementById


<p id="meuParagrafo">Olá, mundo!</p>
          

var paragrafo = document.getElementById('meuParagrafo');
console.log(paragrafo); // Verifique a saída no browser

Usando getElementsByClassName


<ul>
  <li class="item">Item 1</li>
  <li class="item">Item 2</li>
</ul>
          

var itens = document.getElementsByClassName('item');
console.log(itens); // Verifique a saída no browser

Usando querySelector e querySelectorAll 


<div class="container">
  <p>Parágrafo dentro do container.</p>
</div>
          

var container = document.querySelector('.container');
console.log(container); // Verifique a saída no browser

Manipulando Elementos do DOM

Após selecionar os elementos, podemos manipular suas propriedades para alterar o conteúdo ou estilo.

Exemplo: Alterando o Conteúdo de um Elemento


<h1 id="titulo">Título Original</h1>
          

var titulo = document.getElementById('titulo');
titulo.innerText = 'Título Alterado via JavaScript';

Modificando Estilos


titulo.style.color = 'blue';

Eventos e Interação com o Usuário

Os eventos permitem que nossa página responda às interações do usuário, como cliques, teclas pressionadas, e muito mais. 


<button id="meuBotao">Clique em mim</button>
          

var botao = document.getElementById('meuBotao');
botao.addEventListener('click', function() {
  alert('Você clicou no botão!');
});

Exemplo Completo

Vamos criar um exemplo onde interagimos com o DOM para criar uma lista de tarefas simples. 


<input type="text" id="novaTarefa" placeholder="Adicione uma nova tarefa">
<button id="adicionarTarefa">Adicionar</button>
<ul id="listaTarefas"></ul>
          

var inputTarefa = document.getElementById('novaTarefa');
var botaoAdicionar = document.getElementById('adicionarTarefa');
var listaTarefas = document.getElementById('listaTarefas');

botaoAdicionar.addEventListener('click', function() {
  var tarefaTexto = inputTarefa.value;
  if (tarefaTexto !== '') {
    var novaLI = document.createElement('li');
    novaLI.innerText = tarefaTexto;
    listaTarefas.appendChild(novaLI);
    inputTarefa.value = '';
  }
});

Resumo

  • DOM é a representação em árvore de um documento HTML ou XML.
  • Elementos do DOM são objetos manipuláveis com JavaScript.
  • Podemos acessar e modificar propriedades e métodos dos objetos DOM para alterar conteúdo e estilo.

Exercícios

  1. Selecione o elemento <h1> de uma página e altere seu texto para "Olá, DOM!".

    Dica: Use document.querySelector ou document.getElementById para selecionar o elemento <h1>, e a propriedade innerText para alterar o conteúdo.

  2. Altere a cor de fundo de todos os parágrafos (<p>) para amarelo.

    Dica: Você pode usar document.querySelectorAll para selecionar todos os parágrafos e um loop para aplicar a cor de fundo usando style.backgroundColor.

  3. Crie um botão que, ao ser clicado, oculte um elemento na página.

    Dica: Primeiro, selecione o botão e o elemento que deseja ocultar. Em seguida, adicione um evento de clique ao botão usando addEventListener e defina a propriedade style.display do elemento como "none" para ocultá-lo.

  4. Adicione um novo item a uma lista (<ul> ou <ol>) ao clicar em um botão.

    Dica: Use document.createElement para criar o novo item da lista e appendChild para adicioná-lo à lista. Não se esqueça de adicionar um evento de clique ao botão para executar essa ação.

Conclusão

O DOM é uma das peças fundamentais para tornar uma página web interativa. Ao entender como ele funciona, conseguimos acessar e modificar quase todos os elementos do HTML, criando experiências mais dinâmicas para o usuário. Nesta aula de apoio, exploramos desde a estrutura do DOM, suas relações com objetos JavaScript e alguns dos métodos mais comuns para selecionar e manipular elementos.

Lembre-se: cada elemento HTML na página é representado como um objeto que você pode acessar e modificar com JavaScript . As funcionalidades do DOM, combinadas com a lógica e poder do JavaScript, formam a base para construir uma página responsiva e interativa. À medida que avançarmos nas aulas, você verá ainda mais formas de manipular o DOM para responder a eventos, modificar conteúdos e aplicar efeitos visuais.

Esta introdução ao DOM deve ajudá-lo a entender melhor a relação entre JavaScript e os elementos visíveis da página, preparando você para temas mais avançados de manipulação do DOM no futuro.

Continue explorando e experimente novas ideias usando o DOM em seus projetos pessoais. Essa prática é essencial para consolidar o que aprendeu. Bons estudos!
