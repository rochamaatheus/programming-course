Antes de Começar

Vamos iniciar nossa jornada pelo mundo das variáveis em JavaScript! Antes de mergulharmos no código, precisamos preparar nosso ambiente de trabalho criando alguns arquivos básicos.

1. Criando o Arquivo HTML (index.html)

No seu editor de código favorito, crie um novo arquivo chamado index.html. Esse arquivo será a base da nossa página web, onde vamos estruturar o conteúdo que será exibido no navegador.

2. Criando o Arquivo JavaScript (script.js)

Ainda no mesmo local, crie um arquivo chamado script.js. Este será o arquivo onde vamos escrever nosso código JavaScript para dar vida à nossa página.

3. Conectando o JavaScript ao HTML

Para que o código JavaScript funcione em nossa página web, precisamos conectá-lo ao arquivo HTML. Para isso, adicione a seguinte linha de código antes da tag de fechamento </body> no seu index.html: 

<script src="script.js"></script>

Variáveis: O Que São e Como Usá-las

Pense nas variáveis como pequenas caixas onde você pode guardar informações importantes para o seu programa. Elas nos permitem armazenar, modificar e reutilizar dados ao longo do código. Vamos descobrir como elas funcionam em JavaScript!

Por Que Usamos Variáveis?

As variáveis são essenciais porque nos permitem armazenar dados que podem ser usados e manipulados pelo nosso programa. Sem elas, teríamos que escrever os mesmos valores repetidamente, o que não é eficiente. Usando variáveis, seguimos o princípio DRY (Don't Repeat Yourself ou "Não se Repita"), tornando nosso código mais limpo e fácil de manter.

Como Declarar Variáveis

Em JavaScript, existem três palavras-chave para declarar variáveis: var, let e const. Cada uma tem suas particularidades. Vamos entender a diferença entre elas:

  • var: A maneira tradicional de declarar variáveis. Tem escopo global ou de função, o que pode levar a alguns comportamentos inesperados se não for usado com cuidado.
  • let: Introduzido no ES6, permite declarar variáveis com escopo de bloco. É a opção recomendada para variáveis que precisam mudar de valor.
  • const: Também introduzido no ES6, permite declarar constantes, ou seja, variáveis cujo valor não pode ser reatribuído. Também tem escopo de bloco.

Exemplos de Declaração


var cidade = 'Rio de Janeiro';
let temperatura = 30;
const pais = 'Brasil';

No exemplo acima, declaramos três variáveis: cidade é uma string que armazena o nome de uma cidade, temperatura é um número que representa a temperatura atual, e pais é uma constante que guarda o nome de um país.

Manipulando Dados com Variáveis

Além de evitar repetições desnecessárias, as variáveis nos permitem manipular e transformar dados de forma eficiente. Veja este exemplo: 


let largura = 5;
let altura = 10;
let area = largura * altura;
console.log(area); // 50

Neste exemplo, calculamos a área de um retângulo multiplicando a largura pela altura. Se precisarmos alterar as dimensões, basta modificar o valor das variáveis correspondentes.

Sintaxe da Declaração

A sintaxe básica para declarar uma variável é:

let nomeDaVariavel = valor;

Onde let é a palavra-chave usada para declarar a variável, nomeDaVariavel é o identificador que você escolhe, e valor é o dado que você está atribuindo.

Por exemplo:

const pi = 3.14;

Aqui, estamos declarando uma constante chamada pi e atribuindo o valor 3.14 a ela.

Declarando Múltiplas Variáveis

Podemos declarar várias variáveis ao mesmo tempo usando vírgulas:


let produto = 'Caderno', preco = 15, quantidade = 3;

Assim, declaramos três variáveis em uma única linha.

Variáveis Sem Valor Inicial

Também é possível declarar uma variável sem atribuir um valor imediatamente: 


var resultado;
console.log(resultado); // undefined

Neste caso, a variável resultado foi declarada, mas não possui um valor atribuído ainda. Portanto, seu valor é undefined.

Regras para Nomear Variáveis

Ao criar variáveis, precisamos seguir algumas regras:

  • Podem começar com letras, $ ou _.
  • Não podem começar com números.
  • Não podem conter espaços ou caracteres especiais.
  • Sensíveis a maiúsculas e minúsculas (nome ≠ Nome).
  • Não usar palavras reservadas da linguagem.

Exemplos de Nomes Válidos e Inválidos


// Inválidos
let 3dModel;
let class;
let meu-nome;
// Válidos
let $preco;
let _resultado;
let enderecoCliente;

Conceito de Hoisting

Em JavaScript, existe um comportamento chamado hoisting, onde as declarações de variáveis são movidas para o topo do escopo durante a execução, mas as atribuições de valor não são. Isso pode causar resultados inesperados se não entendermos como funciona. 


console.log(saudacao); // undefined
var saudacao = 'Olá!';

Neste exemplo, ao tentar imprimir saudacao antes de declará-la, o resultado é undefined por causa do hoisting da declaração, mas não da atribuição.

Reatribuindo Valores

Podemos mudar o valor de variáveis declaradas com var ou let, mas não com const. 


var contador = 1;
contador = 2; // Ok
const pi = 3.14;
pi = 3.1415; // Erro

No exemplo acima, reatribuímos um novo valor à variável contador, o que é permitido. No entanto, tentar alterar o valor da constante pi resulta em um erro, pois constantes não podem ser reatribuídas.

  1. Crie uma variável para armazenar o nome de sua cidade natal.
  2. Crie uma variável para o ano atual.
  3. Declare uma constante para o número de meses no ano.
  4. Declare uma variável sem valor para a temperatura atual.
  5. Atribua um valor à variável de temperatura que você acabou de criar.
