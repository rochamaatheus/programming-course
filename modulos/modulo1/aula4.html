Introdução aos Números e Operadores

Nesta aula, vamos explorar em detalhes como trabalhar com números e operadores em JavaScript . Compreender esses conceitos é fundamental para realizar cálculos, manipular dados e construir programas mais complexos.

Números em JavaScript

Em JavaScript, o tipo Number é usado para representar tanto números inteiros quanto de ponto flutuante (decimais). Vamos ver alguns exemplos:

Exemplos de Números


var idade = 28;
var gols = 1000;
var pi = 3.14; // Ponto para decimal
var exp = 2e10; // 20000000000

Observações:

  • JavaScript utiliza o ponto ( .) como separador decimal, não a vírgula.
  • Números podem ser representados em notação científica, como em 2e10, que equivale a 2 x 1010.
  • A precisão dos números em JavaScript é de até 15 dígitos significativos.

Operadores Aritméticos

Operadores aritméticos são usados para realizar operações matemáticas básicas. Aqui estão os principais operadores:

Operador Descrição Exemplo Resultado
+ Adição 10 + 5 15
- Subtração 10 - 5 5
* Multiplicação 10 * 5 50
/ Divisão 10 / 5 2
% Módulo (Resto da divisão) 10 % 3 1
** Exponenciação 2 ** 3 8

Exemplos Práticos


var soma = 100 + 50; // 150
var subtracao = 100 - 50; // 50
var multiplicacao = 100 * 2; // 200
var divisao = 100 / 2; // 50
var expoente = 2 ** 4; // 16
var modulo = 14 % 5; // 4

Importante:

  • O operador de adição ( +) também é usado para concatenar strings. Portanto, se um dos operandos for uma string, o resultado será uma string concatenada.

Operadores Aritméticos com Strings

Quando utilizamos operadores aritméticos com strings, o JavaScript tenta converter as strings em números quando possível. Vamos ver alguns exemplos: 


var soma = '100' + 50; // '10050' (concatenação)
var subtracao = '100' - 50; // 50 (conversão para número)
var multiplicacao = '100' * '2'; // 200 (conversão para número)
var divisao = 'Comprei 10' / 2; // NaN (Not a Number)

Observações:

  • A adição com strings realiza concatenação.
  • Operadores como subtração, multiplicação e divisão tentam converter as strings em números.
  • Se a conversão falhar, o resultado será NaN (Not a Number).

Entendendo o NaN (Not a Number)

O NaN é uma propriedade que representa um valor "Não é um Número". Isso ocorre quando uma operação matemática falha (por exemplo, quando tentamos dividir uma string não numérica por um número).

Exemplo com NaN


var numero = 80;
var unidade = 'kg';
var peso = numero + unidade; // '80kg' (string)
var pesoPorDois = peso / 2; // NaN

Para verificar se um valor é NaN, podemos usar a função isNaN(): 


console.log(isNaN(pesoPorDois)); // true

A Importância da Ordem das Operações

Assim como na matemática, a ordem em que as operações são realizadas em JavaScript é importante. A sequência padrão é:

  1. Parênteses
  2. Exponenciação
  3. Multiplicação e Divisão (da esquerda para a direita)
  4. Soma e Subtração (da esquerda para a direita)

Exemplos de Ordem de Operações


var total1 = 20 + 5 * 2; // 30
var total2 = (20 + 5) * 2; // 50
var total3 = 20 / 2 * 5; // 50
var total4 = 10 + 10 * 2 + 20 / 2; // 40

Para alterar a ordem padrão, use parênteses para definir quais operações devem ser realizadas primeiro.

Operadores Aritméticos Unários

Operadores unários são aplicados a apenas um operando. Os mais comuns são o operador de incremento ( ++) e decremento ( --).

Operador de Incremento


var incremento = 5;
console.log(incremento++); // 5 (retorna e depois incrementa)
console.log(incremento); // 6

var incremento2 = 5;
console.log(++incremento2); // 6 (incrementa e depois retorna)
console.log(incremento2); // 6

Observações:

  • incremento++ retorna o valor atual e depois incrementa.
  • ++incremento incrementa primeiro e depois retorna o valor.
  • O mesmo conceito se aplica ao operador de decremento ( --).

Operadores Unários com + e -

Os operadores unários + e - também podem ser usados para converter tipos de dados.

Convertendo Strings em Números


var idade = '28';
console.log(+idade); // 28 (número)
console.log(-idade); // -28 (número)
console.log(+idade + 5); // 33

var possuiFaculdade = true;
console.log(+possuiFaculdade); // 1

Observações:

  • O operador + tenta converter o operando em número.
  • Valores booleanos são convertidos em 1 ( true) ou 0 ( false).
  • Se a conversão não for possível, o resultado será NaN.
