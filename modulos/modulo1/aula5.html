Introdução aos Booleans e Condicionais

Nesta aula, vamos explorar os Booleans e como utilizá-los em estruturas condicionais como if, else e switch em JavaScript. Entender como funcionam as condições é essencial para controlar o fluxo do seu programa.

O Tipo de Dado Boolean

Em JavaScript, o tipo Boolean possui apenas dois valores: true (verdadeiro) ou false (falso). Eles são fundamentais para tomar decisões no código.

Exemplos de Booleans


var possuiGraduacao = true;
var possuiDoutorado = false;

Condicionais: If e Else

As condicionais permitem executar blocos de código com base em certas condições. A estrutura mais básica é o if, que executa o código dentro de suas chaves se a condição for verdadeira. O else é usado para executar código caso a condição do if seja falsa.

Exemplo de If e Else


var possuiGraduacao = true;

if (possuiGraduacao) {
  console.log('Possui graduação');
} else {
  console.log('Não possui graduação');
}
// Retorna 'Possui graduação' e não executa o else

O valor dentro dos parênteses sempre será avaliado como true ou false.

Condicionais: Else If

O else if permite testar múltiplas condições em sequência. Se o if inicial for falso, o programa testará a condição do else if.

Exemplo de Else If


var possuiGraduacao = true;
var possuiDoutorado = false;

if (possuiDoutorado) {
  console.log('Possui graduação e doutorado');
} else if (possuiGraduacao) {
  console.log('Possui graduação, mas não possui doutorado');
} else {
  console.log('Não possui graduação');
}
// Retorna 'Possui graduação, mas não possui doutorado'

Utilizando o Switch

O switch é uma alternativa ao if else quando precisamos verificar se uma variável é igual a diferentes valores. Utilizamos o case para cada valor possível e o break para interromper a execução após uma correspondência.

Exemplo de Switch


var corFavorita = 'Azul';

switch (corFavorita) {
  case 'Azul':
    console.log('Olhe para o céu.');
    break;
  case 'Vermelho':
    console.log('Olhe para rosas.');
    break;
  case 'Amarelo':
    console.log('Olhe para o sol.');
    break;
  default:
    console.log('Feche os olhos');
}
// Retorna 'Olhe para o céu.'

Valores Truthy e Falsy

Em JavaScript, alguns valores são considerados Falsy (avaliados como false) e outros são considerados Truthy (avaliados como true). Conhecer esses valores é importante para evitar comportamentos inesperados.

Valores Falsy

  • false
  • 0 ou -0
  • NaN
  • null
  • undefined
  • '', "", ``

Valores Truthy

  • true
  • Números diferentes de zero (1, -5)
  • Strings não vazias (' ', 'Andre')
  • Objetos e arrays ({}, [])

Operador Lógico de Negação !

O operador de negação ! inverte o valor booleano de uma expressão. Se a expressão é true, ela se torna false e vice-versa.

Exemplos com Operador de Negação


// Exemplos
if (!true) // false
if (!1) // false
if (!'') // true
if (!undefined) // true
if (!!' ') // true
if (!!'') // false

Dica: você pode utilizar !! para verificar se um valor é truthy ou falsy.

Operadores de Comparação

Os operadores de comparação são usados para comparar valores e retornam um valor booleano (true ou false).

Principais Operadores

  • >: Maior que
  • <: Menor que
  • >=: Maior ou igual a
  • <=: Menor ou igual a
  • ==: Igual a (não estrito)
  • ===: Igual a (estrito)
  • !=: Diferente de (não estrito)
  • !==: Diferente de (estrito)

Exemplos de Comparação


10 > 5; // true
5 > 10; // false
20 < 10; // false
10 <= 10; // true
10 >= 11; // false

10 == '10'; // true (não estrito)
10 === '10'; // false (estrito)
10 != '10'; // false (não estrito)
10 !== '10'; // true (estrito)

Operadores Lógicos

Os operadores lógicos permitem combinar expressões e retornam um valor booleano como resultado.

Operador E (&&)

O operador && (E lógico) retorna true se ambas as expressões forem verdadeiras. Caso contrário, retorna false.

Exemplos com &&


true && true; // true
true && false; // false
false && true; // false
'Gato' && 'Cão'; // 'Cão'
(5 - 5) && (5 + 5); // 0
'Gato' && false; // false
(5 >= 5) && (3 < 6); // true

Se ambos os valores forem true, ele retorna o último valor verificado. Se algum valor for false, retorna o primeiro valor false encontrado.

Operador OU (||)

O operador || (OU lógico) retorna true se pelo menos uma das expressões for verdadeira.

Exemplos com ||


true || true; // true
true || false; // true
false || true; // true
'Gato' || 'Cão'; // 'Gato'
(5 - 5) || (5 + 5); // 10
'Gato' || false; // 'Gato'
(5 >= 5) || (3 < 6); // true

Retorna o primeiro valor true que encontrar. Se nenhum valor for true, retorna o último valor avaliado.
