Introdução aos Tipos de Dados

Nesta aula, vamos aprofundar nosso conhecimento sobre os tipos de dados em JavaScript . Entender os diferentes tipos de dados é fundamental para escrever código eficiente e evitar erros inesperados.

Os 7 Tipos de Dados

JavaScript possui sete tipos de dados primitivos e um tipo especial que é o objeto. Aqui estão eles:

  • String: Sequência de caracteres, usada para representar texto.
  • Number: Números inteiros ou decimais.
  • Boolean: Verdadeiro (true) ou falso (false).
  • Undefined: Tipo de uma variável que foi declarada, mas não possui valor.
  • Null: Representa intencionalmente a ausência de valor.
  • Symbol: Novo em ES6, é um valor único e imutável.
  • BigInt: Novo em ES2020, utilizado para representar números inteiros de precisão arbitrária.
  • Object: Coleção de propriedades, usado para estruturar dados complexos.

Todos os tipos acima são considerados primitivos, exceto o Object, que é complexo e permite estruturar dados mais elaborados.

Exemplos de Cada Tipo


var nome = 'André'; // String
var idade = 28; // Number
var possuiFaculdade = true; // Boolean
var time; // Undefined
var comida = null; // Null
var simbolo = Symbol(); // Symbol
var bigNumber = BigInt(123456789012345678901234567890); // BigInt
var objetoQualquer = {}; // Object

Note que primitivos são dados imutáveis . Isso significa que, quando você manipula um valor primitivo, está trabalhando com uma cópia do valor, não com o valor original.

Verificando o Tipo de Dado

Podemos utilizar o operador typeof para verificar o tipo de dado de uma variável: 


var nome = 'André';
console.log(typeof nome); // 'string'

var idade = 28;
console.log(typeof idade); // 'number'

var possuiFaculdade = true;
console.log(typeof possuiFaculdade); // 'boolean'

var time;
console.log(typeof time); // 'undefined'

var comida = null;
console.log(typeof comida); // 'object'

var simbolo = Symbol();
console.log(typeof simbolo); // 'symbol'

var bigNumber = BigInt(12345678901234567890);
console.log(typeof bigNumber); // 'bigint'

var objetoQualquer = {};
console.log(typeof objetoQualquer); // 'object'

Uma curiosidade é que typeof null retorna 'object'. Este é um comportamento legado do JavaScript e pode ser confuso para iniciantes.

Trabalhando com Strings

Strings são cadeias de caracteres usadas para representar texto. Podemos concatenar strings usando o operador + ou utilizar métodos de strings para manipulá-las.

Concatenando Strings

Exemplo de concatenação:


var nome = 'Matheus';
var sobrenome = 'Rocha';
var nomeCompleto = nome + ' ' + sobrenome;
console.log(nomeCompleto); // 'Matheus Rocha'

Quando somamos um número a uma string, o resultado é sempre uma string. Isso acontece devido à coerção de tipos, onde o JavaScript converte o número em string para realizar a operação.

Exemplo com Números e Strings


var gols = 1000;
var frase = 'Romário fez ' + gols + ' gols';
console.log(frase); // 'Romário fez 1000 gols'

Aspas Duplas, Simples e Template Strings

Podemos utilizar aspas simples ( '), aspas duplas ( ") ou Template Strings ( `) para criar strings. Exemplo: 


'JavaScript é "super" fácil';
"JavaScript é 'super' fácil";
"JavaScript é \"super\" fácil";
`JavaScript é "super" fácil`;

Evite misturar tipos de aspas sem a devida escape, pois isso resultará em erros de sintaxe.

Utilizando Template Strings

As Template Strings permitem interpolar variáveis e expressões dentro de strings, facilitando a composição de textos dinâmicos. 


var gols = 1000;
var frase1 = 'Romário fez ' + gols + ' gols';
var frase2 = `Romário fez ${gols} gols`; // Utilizando Template String
console.log(frase2); // 'Romário fez 1000 gols'

Dentro das chaves ${}, podemos colocar expressões ou variáveis.

Trabalhando com Números

O tipo Number em JavaScript é usado para representar tanto números inteiros quanto de ponto flutuante. Também suporta notação científica.

Exemplos de Números


var idade = 28;
var preco = 199.99;
var hex = 0xff; // Hexadecimal
var expo = 2e10; // Notação científica

Podemos realizar operações matemáticas básicas usando os operadores aritméticos: +, -, *, /, % (módulo).

Operações Matemáticas


var soma = 10 + 5; // 15
var subtracao = 10 - 5; // 5
var multiplicacao = 10 * 5; // 50
var divisao = 10 / 5; // 2
var modulo = 10 % 3; // 1

Cuidado ao realizar operações com números e strings, pois o resultado pode não ser o esperado devido à coerção de tipos.

Exemplo de Coerção de Tipos


var resultado = '10' - 5; // 5
var resultado2 = '10' + 5; // '105'

No primeiro caso, a string '10' é convertida em número devido ao operador de subtração. No segundo, o operador de soma concatena os valores como strings.

Trabalhando com Booleanos

O tipo Boolean possui apenas dois valores: verdadeiro (true) ou falso (false). É muito utilizado em estruturas de controle e comparações.

Exemplos de Booleanos


var possuiGraduacao = true;
var possuiDoutorado = false;

Podemos utilizar operadores de comparação para obter valores booleanos:

Operadores de Comparação


var maior = 10 > 5; // true
var igual = 10 === 10; // true
var diferente = 10 !== 5; // true

Undefined e Null

O tipo Undefined é atribuído a variáveis que foram declaradas mas não inicializadas. Já o Null é usado quando queremos intencionalmente indicar que a variável não possui valor.

Exemplos de Undefined e Null


var indefinido;
console.log(indefinido); // undefined

var nulo = null;
console.log(nulo); // null

Lembre-se que undefined e null são diferentes. O primeiro é atribuído pelo JavaScript quando não há valor, enquanto o segundo é atribuído por nós para indicar ausência de valor.

Trabalhando com Symbol

O Symbol é um tipo de dado único e imutável, introduzido no ES6. É muito utilizado para criar propriedades privadas em objetos, evitando conflitos de nomes.

Exemplo de Symbol


var simbolo1 = Symbol('descricao');
var simbolo2 = Symbol('descricao');
console.log(simbolo1 === simbolo2); // false

Mesmo que dois símbolos tenham a mesma descrição, eles são sempre únicos.

Trabalhando com BigInt

O BigInt foi introduzido no ES2020 e permite representar números inteiros de precisão arbitrária, ou seja, números muito grandes que ultrapassam o limite do tipo Number.

Exemplo de BigInt


var numeroGrande = 123456789012345678901234567890n;
console.log(numeroGrande); // 123456789012345678901234567890n

Note o n no final do número, indicando que é um BigInt.

Trabalhando com Objetos

Objetos são estruturas que permitem armazenar dados na forma de propriedades e métodos. São fundamentais em JavaScript e permitem modelar dados complexos.

Exemplo de Objeto


var pessoa = {
  nome: 'André',
  idade: 28,
  possuiFaculdade: true,
  cumprimentar: function() {
    console.log('Olá, ' + this.nome);
  }
};

pessoa.cumprimentar(); // 'Olá, André'

No exemplo acima, criamos um objeto pessoa com propriedades e um método para cumprimentar.
