Introdução aos Arrays

Um array é uma coleção de valores geralmente relacionados. Eles são úteis para armazenar diferentes valores em uma única variável, facilitando a organização e manipulação dos dados.

Exemplo de Array


var comidasFavoritas = ['Pizza', 'Hambúrguer', 'Sushi'];

console.log(comidasFavoritas[0]); // 'Pizza'
console.log(comidasFavoritas[2]); // 'Sushi'

Podemos acessar um elemento específico de um array usando a notação de colchetes [] com o índice do elemento desejado. Lembre-se de que os índices em arrays começam em 0.

Métodos e Propriedades de Arrays

Arrays em JavaScript vêm com uma variedade de métodos e propriedades úteis que nos permitem manipular os dados de forma eficiente.

Manipulando Arrays


var filmes = ['Interstellar', 'Inception', 'Matrix'];

filmes.push('O Poderoso Chefão'); // Adiciona ao final
filmes.pop(); // Remove o último item
console.log(filmes.length); // 3

No exemplo acima, utilizamos o método push() para adicionar um item ao final do array e o método pop() para remover o último item. A propriedade length retorna o tamanho atual do array.

Existem diversos outros métodos disponíveis, como map, filter, reduce e forEach, que exploraremos mais adiante.

Loops: for e while

Loops são estruturas que nos permitem repetir um bloco de código enquanto uma condição for verdadeira. O loop for é um dos mais utilizados em JavaScript.

Loop for

O loop for é composto por três partes: inicialização, condição e incremento. Ele executa um bloco de código até que a condição seja falsa. 


for (var numero = 1; numero <= 5; numero++) {
  console.log('Número: ' + numero);
}
// Irá imprimir os números de 1 a 5

No exemplo acima, inicializamos a variável numero com 1, definimos a condição para que o loop continue enquanto numero <= 5, e incrementamos numero em 1 a cada iteração.

Loop while

O loop while executa um bloco de código enquanto uma condição for verdadeira. 


var contador = 5;
while (contador > 0) {
  console.log('Contagem regressiva: ' + contador);
  contador--;
}
// Irá imprimir de 5 até 1

No exemplo acima, o loop continua executando enquanto contador > 0, decrementando o valor de contador a cada iteração.

Usando Loops com Arrays

Uma das principais utilidades dos loops é iterar sobre arrays, permitindo acessar e manipular cada elemento individualmente. 


var comidasFavoritas = ['Pizza', 'Hambúrguer', 'Sushi', 'Tacos'];
for (var i = 0; i < comidasFavoritas.length; i++) {
  console.log('Eu adoro ' + comidasFavoritas[i]);
}

Neste exemplo, usamos o loop for para percorrer o array comidasFavoritas, acessando cada elemento através de comidasFavoritas[i] e exibindo uma mensagem no console.

O Comando break

O comando break interrompe a execução de um loop, mesmo que a condição ainda seja verdadeira. Isso é útil quando encontramos o que estamos procurando e não precisamos continuar a iterar. 


var comidasFavoritas = ['Pizza', 'Hambúrguer', 'Sushi', 'Tacos'];
for (var i = 0; i < comidasFavoritas.length; i++) {
  console.log(comidasFavoritas[i]);
  if (comidasFavoritas[i] === 'Sushi') {
    console.log('Encontrei meu prato preferido!');
    break;
  }
}

No exemplo acima, o loop irá parar quando encontrar 'Sushi', graças ao uso do break.

O Método forEach

O método forEach é uma forma mais concisa de iterar sobre arrays. Ele executa uma função callback para cada item do array. 


var frutas = ['Maçã', 'Banana', 'Laranja', 'Manga'];
frutas.forEach(function(fruta, indice) {
  console.log('Fruta ' + indice + ': ' + fruta);
});

No exemplo, a função callback recebe os parâmetros fruta (o item atual) e indice (a posição do item no array), permitindo acessar e utilizar essas informações.

Não se Confunda com a Sintaxe

Às vezes, encontramos loops escritos de maneiras menos convencionais. Por exemplo: 


var numero = 0;
var maximo = 5;
for (; numero < maximo;) {
  console.log(numero);
  numero++;
}

Embora funcione, essa forma de escrever um loop pode ser confusa. É importante manter a clareza do código, seguindo as convenções comuns.
